# 📘 Cours JavaScript Complet — BTS SIO SLAM

Un site web interactif en pur HTML/CSS/JS — aucune dépendance, aucun framework.

## ✨ Contenu

- **15 modules** progressifs — des bases jusqu'aux classes, l'async et les APIs browser
